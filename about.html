My Visualisation Journey

I have created this website out of my interest in interactive data visualisation and storytelling. It is a way to cultivate my creativity in design, writing, and using data, drawing on my years of research and data science experience, but not being constrained by these formal training.

I use mainly Python for data processing and statistical analysis, and one of its libraries Altair Vega for creating visualisation in the first two pieces - Mapping UK diversity and Mountain Rescues.

I am currently learning D3 library in the hope of creating more customized and individualised visualisation, and hopefully my next piece would make use of it. The logo of this website is created with it.

Other than these particular tools, I mainly use pure HTML, CSS, Javascripts to create this site for the purpose of doing web design from scratch.
